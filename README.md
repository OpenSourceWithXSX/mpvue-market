## wxapp-market

> mpvue小程序营销组件，原版[wxapp-market](https://github.com/o2team/wxapp-market)

## 支持营销玩法

- 大转盘
- 刮刮乐
- 老虎机
- 水果机
- 摇一摇
- 手势解锁


## 预览

``` bash
1. git clone
git clone https://github.com/wkl007/mpvue-market.git

2. 安装依赖
cd mpvue-market && npm install

3. 启动程序
npm run dev

4. 预览
打开微信开发者工具，新建项目，将目录指向 /dist/wx 即可
```
